# Patternfly docs test
Built with [Eleventy](https://www.11ty.io/), Gulp and Sass.

## Installation
Make sure you have at lealst Node 10.x.x and run:

```shell
npm install
```

## Running Local Server
```
npm start
```
This will start eleventy and a watch process that will reload the site if `*.njk`, `*.scss`, or `*.es6` files are updated.
